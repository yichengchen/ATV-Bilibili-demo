# ADRs

Use an Architecture Decision Record when a change introduces a durable technical decision.

Typical cases in this repository:

- Playback pipeline changes
- New player plugins or plugin lifecycle rules
- Request signing or authentication changes
- New persistence formats or settings migrations
- Cross-module navigation or factory changes
- New third-party dependencies

Copy [`docs/adr/0000-template.md`](./0000-template.md) to the next available number and link it from the related spec and pull request.
